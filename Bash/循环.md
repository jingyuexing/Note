# 循环
```bash
i的初始值为0,i循环十次,每次循环加一
for (( i = 0; i < 10; i++ )); do
    # 每次循环的时候输出"你好"
    echo "你好"
done
```
***for...in循环***
```bash
for i in 1 2 3 4 5 6 7 8 12 33 42 56 99; do #列举所有的元素1 2 3 4 5 6 7 8 12 33 42 56 99
    #statements
    echo "${i}"
done
```

***while***
```bash
while(($i<=10)); do
    #statements
    echo "${i}"
    let "i++"
done
```
